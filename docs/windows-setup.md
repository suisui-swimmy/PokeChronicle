# Windows Setup

Use PowerShell from the repository root.

```powershell
npm install
npm run dev
```

For validation:

```powershell
npm run test
npm run build
npm run preview
```

If a Vite or Vitest command fails with a sandbox-style path access error such as `Cannot read directory "../.."`, treat it as an environment issue first and rerun from a normal PowerShell session.
